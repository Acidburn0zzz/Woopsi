About Woopsi - Features

  • Multiple screen support
  • Full featured windows, with depth sorting, close gadgets, etc
  • Standard UI components such as textboxes and buttons
  • Smooth scrolling text viewer
  • Smooth scrolling "SuperBitmap" gadget to display large bitmaps in small windows
  • Set of drawing functions for the SuperBitmap gadget, including rectangle and circle drawing
  • Monochrome and colour font support
  • Support for any standard PALib-style 16-bit bitmap font
  • Full featured event system
  • Optimised UI component redrawing
  • Hardware-accelerated rectangle drawing
  • Object-orientated design for easy integration into other C++ software
  • Simple API for rapid GUI creation
  • 16-bit display
  • Appearance mimics AmigaOS 3.0
