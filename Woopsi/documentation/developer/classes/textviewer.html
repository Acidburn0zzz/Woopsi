Gadget Developer Guide - Classes - TextViewer

Overview

The TextViewer gadget provides a simple way to display a large quantity of text. It can be scrolled vertically using the stylus. Dragging the gadget up or down scrolls the text smoothly to match. The gadget automatically wraps text at line returns or spaces to fit within its dimensions.

Rather than create a large bitmap and move that up and down, the TextViewer draws each row of text to a small, immobile bitmap as that row scrolls into the screen. Scrolling is achieved using the DS' hardware to move the existing rows up or down. This means that the TextViewer has relatively little memory overhead - roughly 160K for 1MB of text. The TextViewer can theoretically scroll through more text than the DS has memory.
