Developer Guide - Running the GUI

If you were to take the code from the Creating a Simple GUI section, insert it into a project and run it, you'd find that nothing happened. We're missing two vital steps - we need to draw the GUI, and then we need to ensure that Woopsi is running. To do that, we just need to call two of Woopsi's functions, like this:

// Draw the GUI
woopsi.draw();

// Infinite loop to keep the program running
while (1)
{
	woopsi.play();
	PA_WaitForVBL();
}

The code below provides a complete version of the GUI we've defined that you can copy into your "main.cpp" file and run:

// Includes
#include <PA9.h>
#include "woopsi.h"

int main()
{
	PA_Init();    // Initializes PA_Lib
	PA_InitVBL(); // Initializes a standard VBL

	// Initialise the screen
	PA_Init16bitBg(0, 0);

	// Create a simple GUI
	Woopsi woopsi;
	Screen* screen = woopsi.addScreen("My Screen");
	Window* window = screen->addWindow(20, 20, 100, 100, "My Window", false, true);
	Button* button = window->addButton(5, 5, 60, 60, "My Button"));

	// Draw the GUI
	woopsi.draw();

	// Infinite loop to keep the program running
	while (1)
	{
		woopsi.play();
		PA_WaitForVBL();
	}

	return 0;
}
